# K-Startup API 관찰 기록

실제 API 호출을 통해 확인한 사실만 기록한다. 인증키와 민감한 요청 URL은
기록하지 않는다. 추측은 "미확인"으로 명시한다.

## 호출 기록

### 2026-07-21 — 첫 실제 호출

- 기능: `getAnnouncementInformation01`
- 요청 변수: `page=1`, `perPage=3`, `returnType=json` (ServiceKey 제외)
- 결과: HTTP 200, 정상 JSON. 원본: `data/raw/kstartup_announcements_20260721T060341Z_p1.json` (Git 제외)
- 전체 건수: 29,505 (진행·종료 공고가 섞여 있는 것으로 보이나 미확인)

#### 공식 가이드와 다른 점

- 신청 제외 대상 필드의 실제 표기는 **`aply_excl_trgt_ctnt`** (가이드 표기 후보였던
  `aply_exclt_trgt_ctnt`와 다름)
- 가이드에 없는 필드 존재:
  - **`id`**: 표본에서 1, 2, 3 — 페이지 내 순번으로 보임. **안정적 식별자로 사용 금지**
    (식별자는 `pbanc_sn` 사용)
  - **`biz_aply_url`**: 표본 3건 모두 `null`, 용도 미확인
- `pbanc_sn`은 문자열이 아닌 **숫자(number)** 타입

#### 값 형태 관찰 (정규화 설계에 중요)

- 날짜: `pbanc_rcpt_bgng_dt`/`pbanc_rcpt_end_dt`는 **`YYYYMMDD` 문자열** (시각 없음,
  예: `"20260720"`). 종료 시각 정책(Asia/Seoul 그날 끝) 필요
- `supt_regin`: 짧은 지역명 (`"전국"`, `"경기"`)
- `biz_enyy`: **쉼표로 구분된 다중 값** (예: `"예비창업자,1년미만,2년미만,3년미만"`) —
  단일 조건 문자열이 아니라 허용 범주 목록으로 보임
- `aply_trgt`, `biz_trgt_age`도 쉼표 구분 다중 값 (예: `"청소년,대학생,일반인"`)
- `rcrt_prgs_yn`: `"Y"` (표본은 전부 Y — `"N"` 값 형태는 미관찰)
- `intg_pbanc_yn`: `"Y"`/`"N"` 관찰
- `sprv_inst`: `"공공기관"` (기관 유형 문자열)
- URL 필드에 **프로토콜이 없는 값 존재** (`biz_gdnc_url: "www.kised.or.kr/..."`,
  notion.site 주소 등) — 정규화 시 스킴 보정 필요
- 본문 텍스트에 `\r\n` 줄바꿈, HTML 엔티티(`&apos;`), 원천 인코딩 손상으로 보이는
  문자(`마을?청년`) 존재
- `aply_mthd_eml_rcpt_istc`에 base64 형태 문자열 관찰
  (`"...=="` 패턴, 44자) — 이메일 주소를 암호화/인코딩해 제공하는 것으로 추정되나
  **미확인**. 디코딩을 시도하거나 의미를 단정하지 않는다
- `aply_mthd_onli_rcpt_istc`는 설명 텍스트가 아니라 **URL**이 들어오는 경우 관찰

### 2026-07-21 — 표본 확장 관찰 (100건, page 1~5 × perPage 20)

- `supt_regin`에 **결합 지역 표현** 존재: `"전남광주"` (4건). 그 외 관찰 값:
  전국/서울/경기/강원/대구/대전/전북/충남/제주 — 모두 짧은 시도명
- `rcrt_prgs_yn="N"` 11건 관찰 (첫 호출 표본에서는 Y만 있었음)
- `biz_trgt_age`는 3개 고정 토큰의 조합으로만 나타남:
  `만 20세 미만` / `만 20세 이상 ~ 만 39세 이하` / `만 40세 이상`
  (표본에서 4가지 조합). 부분 목록이 청년 전용 등 연령 제한을 표현하며,
  본문의 세부 상한(예: "45세 이하")은 필드로 표현되지 않음
- `biz_enyy` 토큰은 첫 관찰과 동일한 7종만 나타남
  (`예비창업자`, `1/2/3/5/7/10년미만`)
- `aply_trgt` 토큰도 관찰된 7종만 나타남
  (청소년/대학생/일반인/대학/연구기관/일반기업/1인 창조기업)
- `biz_gdnc_url` 프로토콜 누락이 흔함 (100건 중 64건)
- 페이지네이션: page 1~5 연속 조회에서 중복·누락 없이 100건 수신
  (`pbanc_sn` 기준)

### 2026-07-21 — 상세페이지 대조 관찰 (표본 2건: 178605, 178602)

크로스체크로 실제 상세페이지를 열어 API 데이터와 대조한 결과, 표시 필드
(제목·지역·접수기간·업력·연령·대상·제외대상)가 모두 일치했다. 추가 관찰:

- **접수 마감 시각은 상세페이지에만 있다** (예: "~ 2026-07-31 16:00",
  "~ 2026-07-22 18:00"). API는 날짜(`YYYYMMDD`)만 제공하므로 실제 마감
  시각이 이른 공고는 마감 후 몇 시간 동안 모집 중으로 보일 수 있다
- **첨부파일(PDF/HWP/JPG 등) 다운로드 링크는 상세페이지에 존재**한다
  (API 응답에는 없음). 페이지에 "공공누리 제1유형 저작물, 공공데이터포털
  개방" 문구가 명시되어 있다 — 8단계 첨부파일 수집 설계 시 참고
- 사이트의 "창업업력: 전체", "대상연령: 전체" 표기는 API의 전체 구간
  목록(7개 업력 토큰 / 3개 연령 토큰)과 대응된다
