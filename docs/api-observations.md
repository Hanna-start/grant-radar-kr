# K-Startup API 관찰 기록

실제 API 호출을 통해 확인한 사실만 기록한다. 인증키와 민감한 요청 URL은
기록하지 않는다. 추측은 "미확인"으로 명시한다.

## 상태: 아직 실제 호출 전

첫 실제 호출 후 아래 항목을 채운다.
